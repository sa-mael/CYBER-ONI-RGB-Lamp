# CYBER-ONI-RGB-Lamp  
A custom-designed, 3D-printed RGB cyber-aesthetic lamp inspired by Oni masks and cyberpunk design.  
The lamp is fully printable, supports RGB LED strips, and includes a modular internal frame for easy assembly.

##  Features
- 3D-printed cyber-mask structure (STL included)
- Fully working RGB lighting (5V LED strip)
- Modular internal skeleton for wiring
- 32 cm tall / 40 cm long / 20 cm wide
- PTG body + transparent light diffuser
- GitHub-ready source files (STL + code)
- Open-source for makers and hobbyists
